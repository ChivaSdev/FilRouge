## Ce projet est un fil rouge permettant à l'utilisateur de jouer avec un ami ou contre l'IA à plusieurs jeux tels que :

- Le jeu de Nim
- Le Morpion 
- Une variante du morpion (avec indices) dédié aux joueurs les plus débutants.
